Quick Reference

Constructor

$('selector').turn([options]);

Creates an instance of turn.

Parameters:

  • options
    • page (type: number, default: 1)
    • shadows (type: boolean, default: true)
    • duration (type: number, default: 600)
    • acceleration (type: boolean, default: true)

Calling methods or properties

$('selector').turn(property | method [, parameters]);

turn.js

I decided to develop turn.js, because it was neccesary as a part of an app for magazines. I spent a while working in the math for this effect. Being inspired by the effect of iBooks and Maps for iOS, I created two classes: one for manipulating the overall publication and another for each individual page. The last one is completely independent, in that it offers an API for creating its own implementations.
